# 연산 연습 재도전·응원 기능 (2025-03-16)

## 변경 요약

- **파일**: `math/math_drill.html`
- **내용**: 틀렸을 때 박지훈 응원 모달, 다시 풀기 후 만점 시 응원 메시지

## 흐름

1. **틀렸을 때 (score < 100)**
   - 박지훈 모달 표시
   - "N문제 맞췄어요! (N/Total)"
   - "그것만으로도 대견해요! 👏"
   - "오빠랑 다시 풀어보자!"
   - [다시 풀기] 버튼

2. **다시 풀기 클릭**
   - 입력 초기화, 정답/오답 표시 제거
   - 같은 세트 재도전

3. **재도전 후 만점 (score === 100)**
   - confetti + 저장
   - "열심히 노력해서 결국 해냈어! 라희 대단해! 🎉" alert

4. **첫 시도 만점**
   - 기존과 동일 (confetti + 저장)
