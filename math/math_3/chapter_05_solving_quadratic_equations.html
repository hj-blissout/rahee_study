다 이 방정식을 풀기 위해서였답니다! 마지막 퍼즐을 맞춰봐요 🧩

1단계: 이것만은 기억하자! 🌱

방정식의 답(해)이란?

중1, 중2 때 배운 방정식 기억나시나요?
등호(=) 양쪽이 진짜로 똑같아지게 만드는 x값을 찾는 게 방정식의 목표예요.

x - 3 = 0 일 때, x는 얼마일까요?

👉 톡 눌러서 확인!
정답: x = 3
(3을 넣어야 3 - 3 = 0 이 성립하니까요!)
2단계: 개념 쏙쏙! 💡

곱해서 0이 된다는 건 기적!

어떤 두 수(A와 B)를 곱했는데 0이 나왔어요.
A × B = 0 이 시나리오가 가능하려면 무조건! 적어도 둘 중 하나는 0이어야 해요.
(x - 2)(x - 3) = 0

이 뜻은?
앞 괄호가 0 이거나,
뒤 괄호가 0 이다!
정답 찾기

앞이 0 되려면: x = 2
뒤가 0 되려면: x = 3
그래서 답이 2개예요!
3단계: 한 걸음 더! 👟

본격적인 풀이 시작!

이제 4단원에서 배운 인수분해를 써먹을 차례예요!
모든 이차방정식 풀이의 시작은 우변을 '= 0' 으로 만들고 왼쪽을 인수분해하는 거랍니다.
x² - 5x + 6 = 0 풀기!

1. 인수분해 하기: (더해서 -5, 곱해서 6)

👉 (x - 2)(x - 3) = 0

2. 앞/뒤 괄호가 0이 되는 값 찾기

정답: x = 2 또는 x = 3

4단계: 나도 할 수 있다! 💪

근의 공식 (치트키 등장! ⚡️)

"선생님! 도저히 인수분해가 안 되는 숫자는 어떡하죠?"
그럴 때를 대비해서 옛날 수학자들이 만능 공식(치트키)을 만들어 뒀어요.
ax² + bx + c = 0 일 때,

x =
-b ± √(b² - 4ac)
2a


노래로 외우면 평생 안 까먹는 전설의 근의 공식이에요!
5단계: 자신감 뿜뿜! 👑
